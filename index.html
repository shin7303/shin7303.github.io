プロジェクト名 2: ポートフォリオサイトジェネレーター (Python)

Markdownファイルから静的なポートフォリオサイトを生成するPythonスクリプトです。 設定ファイルで基本情報やプロジェクト情報を記述し、スクリプトを実行するだけでHTMLが生成されます。 テンプレートエンジンにはJinja2を使用しました。

コードスニペット (Python)

# Pythonスクリプトの例
import os
from jinja2 import Environment, FileSystemLoader

def generate_portfolio(config_data, template_dir='templates', output_dir='output'):
    """
    ポートフォリオサイトを生成する関数
    """
    if not os.path.exists(output_dir):
        os.makedirs(output_dir)

    env = Environment(loader=FileSystemLoader(template_dir))
    template = env.get_template('portfolio_template.html')

    html_content = template.render(data=config_data)

    output_path = os.path.join(output_dir, 'index.html')
    with open(output_path, 'w', encoding='utf-8') as f:
        f.write(html_content)
    print(f"ポートフォリオが {output_path} に生成されました。")

if __name__ == '__main__':
    # ダミーデータ
    my_config = {
        'name': 'Taro Yamada',
        'introduction': 'Pythonista and Web Developer.',
        'projects': [
            {'title': 'Project Alpha', 'description': 'An amazing project.'},
            {'title': 'Project Beta', 'description': 'Another cool project.'}
        ]
    }
    generate_portfolio(my_config)

説明

このPythonスクリプトは、Jinja2テンプレートエンジンを使用してHTMLファイルを生成します。 generate_portfolio関数が主要な処理を行い、設定データとテンプレートからHTMLを構築し、指定された出力ディレクトリに保存します。 これにより、コンテンツの更新が容易になります。
